# Meu Primeiro PR

Este é um repositório de prática para aprender o fluxo de Pull Requests no GitHub.

## Sobre

Este projeto foi criado para eu treinar como enviar contribuições no GitHub: criar uma branch, fazer uma alteração, e abrir um Pull Request.

## Como contribuir

1. Crie uma branch
2. Faça sua alteração
3. Abra um Pull Request decrevendo a mudança
